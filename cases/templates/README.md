# Generic Deck Templates

These templates are neutral starting points for `ppt-design`.
They are structured for business, strategy, project, and proposal decks rather than for explaining the skill itself.

Use them when you want:

- a reusable deck skeleton with no fixed topic
- a decision-oriented page sequence before writing real content
- a neutral base before adapting the deck to a real business, product, research, or brand story

Included files:

- `five-slide-generic.md`
- `five-slide-generic.zh.md`
- `ten-slide-generic.zh.md`
- `geometry-stress-test.md`
- `geometry-stress-test.zh.md`

They are intentionally topic-free. Replace the placeholders with your own content, then let the skill choose a style and layout prototype per slide.
The default sequence now follows a practical deck rhythm:

- cover
- context
- comparison
- metric
- thesis
- synthesis
- process
- risks
- milestone map
- decision / closing

Use the geometry stress-test files when you want to validate how the skill handles:

- strict tables
- org charts
- framework diagrams
- responsibility matrices
- process maps where box size and position carry meaning
