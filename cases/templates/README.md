# Generic Deck Templates

These templates are neutral starting points for `ppt-design`.

Use them when you want:

- a reusable deck skeleton with no fixed topic
- a clean page structure for style testing
- a neutral base before writing real content

Included files:

- `five-slide-generic.md`
- `five-slide-generic.zh.md`
- `ten-slide-generic.zh.md`

They are intentionally topic-free. Replace the placeholders with your own content, then let the skill choose a style and layout prototype per slide.
